**AI Climate-Resilient Farming Platform**

An AI-powered agricultural decision-support platform designed to improve climate resilience in UK farming. The system integrates satellite imagery, weather forecasts, and soil/climate datasets to detect crop stress and recommend adaptive farming strategies such as irrigation optimisation and planting adjustments.

This project explores how machine learning and remote sensing can support sustainable agriculture, improved productivity, and UK food security under increasing climate variability.

**Project Overview**

Agriculture in the UK is increasingly affected by extreme weather events such as droughts, heatwaves, and irregular rainfall patterns. Farmers often rely on fragmented tools that do not integrate multiple data sources into actionable insights.

This project addresses that challenge by developing an AI-powered predictive platform that:

Detects crop stress early using satellite and environmental data

Predicts climate-related risks to crop health

Recommends adaptive farming strategies

Supports data-driven agricultural decision making

**Objectives**

The main objectives of this project are:

Analyse the impact of climate change on UK agricultural productivity

Develop machine learning models to detect crop stress

Integrate satellite, weather, and soil data sources

Generate actionable recommendations for irrigation and planting

Evaluate the potential of AI to improve food security and sustainability

**Key Features**

Crop stress prediction using AI and remote sensing

Adaptive farming recommendations
